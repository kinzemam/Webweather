# 🌤️ WebWeather

A responsive weather web application built using **Node.js**, **Express.js**, **EJS**, and the **OpenWeather API**. Users can search weather information using a ZIP/Postal code and country code to get real-time weather conditions for locations around the world.

## 📌 Live Demo

🔗 Live Website: https://webweather-3on3.onrender.com

Visit the deployed application to check real-time weather information for locations worldwide using ZIP/Postal codes and country codes.


## 🚀 Features

* Search weather by ZIP/Postal code and country code
* Real-time weather data using OpenWeather API
* Displays:

  * Current temperature
  * Feels-like temperature
  * Minimum and maximum temperature
  * Humidity
  * Atmospheric pressure
  * Sunrise and sunset times
  * Weather condition and icon
* Dynamic weather information rendering with EJS
* Responsive and modern UI
* Deployed on Render

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Webweather.git
```

2. Navigate to the project directory

```bash
cd Webweather
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node index.js
```

5. Open your browser

```text
http://localhost:3000
```

## 🌍 API Used

This project uses the OpenWeather API to fetch:

* Current weather data
* Location information
* Sunrise and sunset timings
* Atmospheric conditions
